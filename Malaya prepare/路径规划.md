## 一、最终 3 条博士出路（优先级从高到低）

1. **T1 首选：新加坡 NUS/NTU 全奖 PhD（月薪≈3000 新币，免学费，全包）**｜最难、含金量最高，回内地高校入职最优
2. **T2 备选：香港八大全奖 + HKPFS 港府博士奖学金（月薪≈1.8w 港币）**｜申请窗口期固定、名额充足，容错高
3. **T3 保底：马来亚本校全奖 PhD（每月 3000 马币津贴，全免学费）**｜本校硕升博导师熟、门槛更低，保底兜底

## 二、分阶段精准时间线（9 月入学，学制 1.5 年授课硕，2028 年 9 月博士入学）

### 【第一阶段：研一上｜入学 0-6 个月（2026.9–2027.2，重中之重：拿 GPA + 锁定 RA）】

**核心目标：GPA 稳住 3.7+/4.0（均分 85+）、拿下带薪 RA（公费出访课题组）、敲定细分方向：医学影像分割 + 统计建模**

1. **选课策略**
    
    优先选：机器学习高阶、医学图像分析、数理统计、深度学习，**只冲 A/A-，避开水课**；所有任课老师课后主动交流科研，铺垫推荐信人选（至少锁定 2 位授课教授）。
2. **RA 申请（入学 1 个月启动，10 月集中投递校内 AI / 数字医疗实验室 RA）**
    
    - 定向：UM 医学院 + 计算机系联合医疗影像课题组（公费出国参会 1–2 周的项目组），邮件模板：本科软件工程 + Python 项目 + 意向医疗影像，可无偿先打杂 1 个月转正带薪 RA（月薪 1800–2800 马币）。
    - 拿到 RA=**现成项目 + 免费数据集 + 导师指导 + 公费短期访学资源**，是发 SCI 唯一捷径。
    
3. **个人工作流落地**：Obsidian 文献库 + Xournal 公式 / 论文批注，按月归档课题文献，提前攒论文素材。
4. **语言：全英授课可豁免新港博士雅思，保留在读证明即可，不用额外考雅思**

### 【第二阶段：研一下｜2027.3–2027.8（科研攻坚：做实验 + 写 SCI 初稿 + 首轮套磁）】

1. **科研主线（RA 全职干活，8 个月搞定一篇 SCI 二区）**
    
    - 3–5 月：依托课题组医疗影像数据改进模型（CNN/Transformer 轻量化 + 统计指标优化 AUC），跑全量实验、整理实验记录；
    - 6–8 月：全英文写 SCI 初稿，导师返修打磨，**8 月底定稿投稿 JCR Q2 / 中科院二区统计 / 医学 AI 期刊**（审稿周期 3–5 个月，次年年初录用）。
    
2. **博士前期筹备**
    
    - 筛选新港 30 位匹配导师（近 3 年论文：医疗影像 + 机器学习），分梯队：冲刺（NUS/NTU 顶组）、稳妥（港中文 / 港城）、保底（UM 本校）；
    - 首轮套磁短邮：简历 + RA 项目简介 + 论文在投摘要，试探导师名额。
    
3. **GPA 收尾：剩余课程尽量 A，硕士最终成绩单≥3.65 是新港全奖底线。**

### 【第三阶段：研二上｜2027.9–2027.12（博士申请黄金窗口期，HKPFS + 新港首轮截止）】

> 港校主轮 12.1 截止、HKPFS 港府奖学金 12.1 截止；新加坡 NUS/NTU 首轮 4 月截止、滚动录取。

1. **论文进度**：SCI 返修中 / 录用接收，录用函是套磁绝杀筹码；没录用用「Accepted 待见刊」证明。
2. **全套申请材料定稿**
    
    - CV：RA 经历 + 公费出访参会 + 论文 + 课程高分；
    - RP 研究计划书（4–6 页）：紧扣目标导师方向（医疗影像统计建模），针对性修改每个导师版本；
    - 2 封教授推荐信：RA 导师 + 高分授课老师（硕士最强 2 位推荐人）。
    
3. **集中套磁 + 网申**
    
    - 9–10 月：密集套磁，导师口头有意向后立刻提交网申；
    - 11–12 月：香港全部院校提交 HKPFS + 常规博士申请；新加坡分批次投递。
    

### 【第四阶段：研二下｜2028.1–2028.8（面试 + 收全奖 offer + 硕士毕业）】

1. **1–4 月：新港集中线下面 / 线上面试，陆续下发全奖录取**，新加坡部分院系 4 月关闭首轮奖学金通道；
2. **5–7 月：完成硕士结课、毕业论文（可沿用 SCI 课题内容，极大省力），拿硕士学位证；**
3. **8 月：确认博士 offer、办签证，2028.9 正式入读全奖 PhD。**

## 四、备选容错方案（万一一区 SCI 延期录用）

1. **保底方案 1：先发 Scopus Q1（马来期刊好中）保底申 UM 本校全奖博士，次年拿着 SCI 再转学新加坡**；
2. **保底方案 2：硕士毕业延后半年（延期结业），等 SCI 录用再申下一轮新港博士，全奖成功率翻倍。**

## 五、避坑关键

1. **不选纯应用型课程**，全部往统计、医学图像处理、深度学习靠拢；
2. **RA 越早进组越好，不要自费找数据做课题**，公费课题组资源 = 省钱 + 省时间；
3. **套磁不要群发邮件，每个导师单独修改 RP 和邮件内容**。
